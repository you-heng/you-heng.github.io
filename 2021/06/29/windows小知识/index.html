windows小知识

发表于 2021-06-29 更新于 2021-07-05 阅读次数：

本文介绍在使用windows时所遇到的问题和解决办法。

windows 中的回车换行

在 windows 中的回车换行是\r\n

如果只用\r可能会出现一些 bug

比如：我用 PHP 代码生成一个文件 1.bat，在文件中写入内容，其中包含了\r，生成的文件就会出现问题，问题如下：

异常的文件是无法执行的

在 windows 下需要注意这个问题

win10 开启热点手机无法连接网络

因为我使用了 Hyper-V 虚拟机，虚拟机设置连接网络时需要以太网设置共享网络给虚拟机，所以只需要把网络共享给 wifi 的连接即可

win10虚拟wifi热点

一般只有在笔记本电脑上才有热点分享，我们可以开启一个虚拟热点使用

ssid是热点的名字，key是热点的密码

执行命令后在设置->状态->更改适配器里面会出现一个新的本地连接名字就是JayFan

点击以太网右键属性->共享

勾选允许其他网络用户通过此计算机的internet连接来连接，下拉框选中刚刚创建的网络

然后手机和平板就可以连接这个热点了
