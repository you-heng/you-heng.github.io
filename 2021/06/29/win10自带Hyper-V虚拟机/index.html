win10自带Hyper-V虚拟机

发表于 2021-06-29 更新于 2021-07-05 阅读次数：

本文介绍win10自带的Hyper-V虚拟机的使用方法。

在控制面板中进入程序

点击启动或关闭 windows 功能

勾选 Hyper-V

等一会儿

立即重新启动

打开 Hyper-V 管理器

windows 管理工具里面的 Hyper-V 管理器

进入管理页面

开始创建虚拟机

可以直接在这里快速新建

然后开始选择安装信息，选择下载好的镜像文件

也可以在管理器中新建

根据提示直接下一步下一步即可

安装系统

虚拟机创建好开始安装系统了

选择直接安装 CentOS 即可

等待一会儿

选择语言

选择磁盘

设置 root 密码

安装完成选择重启

进入虚拟机系统

使用账号密码登录

查看连接状态

使用命令nmcli d查看，红色表示未连接

使用命令nmtui进入网络设置，进到Edit a connection中

回车进入 eth0 中

回车选择这个选项，然后保存

保存后就回到这个页面，选择退出

再退出

然后再用nmcli d命令查看网络是否连接成功

wifi 下配置网络连接

此时发现虚拟机无法连接网络，ping 不通，需要设置一下虚拟交换机

新建虚拟交换机

设置完成后进入系统 输入命令 进入文件编辑 vi /etc/sysconfig/network-scripts/ifcfg-eth0

esc 然后保存退出:wq

重启网络配置service network restart

查看 ipip a s

ping 百度

本地连接配置网络连接

配置了 wifi 状态下的网络连接，发现在本地网络还是无法连接，需要再新建一个虚拟网络交换机

这里选内部的

设置名称等

添加成功后在本机的网络共享中心可以看到有未识别的网络

网络连接中也会出现一个未识别的网络，已经创建了虚拟设备，但还无法连接网络

在网络与共享中心下面点击现有的以太网 2

选择属性

在共享里面勾选’允许其他网络用户通过此计算机的 Internet 连接来连接’并在下方’家庭网络连接’中选择刚刚创建好的虚拟交换机

以管理员身份运行 cmd

在命令行中输入route print 查看 ipv4 路由表中找到关于 192.168.137.1 的信息

然后就可以开始设置虚拟机了

在网络适配器中选中刚刚创建好的虚拟交换机

然后就可以开始重启虚拟机了，进入系统 输入命令 进入文件编辑 vi /etc/sysconfig/network-scripts/ifcfg-eth0 和 wifi 配置同样的设置，如果设置过了可以检查一下

配置完成 ping 一下是否成功

wifi 和本地连接网络的切换

配置好后根据网络切换网络适配器即可

检查点

在 Hyper-V 中检查点就是 VM 的快照

创建检查点

使用检查点

当前系统的设置

Hyper-V 的设置

问题

  1. 因为 Hyper-v 设置网络连接时需要以太网把网络共享给虚拟机，所以 win10 在开启热点时可能会出现无法连接的问题，需要将以太网共享给热点的本地连接。好像使用 VM 虚拟机好像会出现 IP 冲突的问题，wifi 同样无法连接网络，一时之间也不知道哪个更方便一些。
