github在线阅读源码

这里介绍三种方式

github1s

作者对其解释是只需要一秒就可以用VS code方式来查看github代码

Open in VS Code插件

Chrome Open in VS Code扩展

以 Chrome 为例，github1s作者已经为此添加了扩展程序，右键选择 Open in VS Code即可转换为VS Code

Chrome surf扩展

surf也是Chrome插件，安装后会直接在github页面生成一个surf按钮，点击即可转换为VS Code
