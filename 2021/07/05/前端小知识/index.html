本文介绍在前端中所遇到过的问题和解决方法

html

移动端限定屏幕缩放

在H5端有时候不能让网页进行缩放，因为很容易导致误触。用户体验很不好

通过maximum-scale和minimum-scale限定目录缩放比例，通过user-scalable限制用户对屏幕进行缩放

css

站点的公共样式、公共方法和公共参数等

消除不同浏览器默认样式的不一致性

box-sizing属性

box-sizing是用于告诉浏览器如何计算一个元素的总宽度和总高度

box-sizing:content-box; 标准盒模型，内边距外边距不算在width中。真实宽度=width+内边距+外边距

box-sizing:border-box; IE盒模型，内边距外边距都算在width中。 真实宽度=width(包括内边距和外边距)

css伪类选择器

伪类是添加到选择器的描述性词语，指定要选择的元素的特殊状态。伪类需要按照顺序写，否则会不生效

如：超级链接拥有四个特殊状态

伪类 意义
a:link 没有被访问的超级链接
a:visited 已经被访问过的超级链接
a:hover 正被鼠标悬停的超级链接
a:active 正被激活的超级链接(按下案件但是还没有松开的键)

css3新增伪类

伪类 意义
:empty 选择空标签
:focus 选择当前获得焦点的表单元素
:enabled 选择当前有效的表单元素
:disabled 选择当前无效的表单元素
:checked 选择当前已经勾选的单选按钮或者复选框
:root 选择根元素，即标签

相邻选择器和通用选择器

序号选择器

举例 意义
:first-child 第一个子元素
:last-child 最后一个子元素
:nth-child(3) 第三个子元素
:nth-of-type(3) 第三个某类型子元素
:nth-last-child(3) 倒数第三个子元素
:nth-last-of-type(3) 倒数第三个某类型子元素

:nth-child()可以写成an+b的形式，表示从b开始每a个选一个，注意不能写为b+an。例：.box p:nth-child(3n + 2){} 表示从2开始每3个选一个

:nth-child()也可以使用odd，等价与(2n+1)。例：:nth-child(odd) 表示选择奇数

:nth-of-type()使用方法.box p:nth-of-type(3)。表示不管p标签中有没有其他标签，选择box里面的第三个p标签

其他方法也是差不多用法，:first-child数兼容到IE7，其他兼容到IE9

属性选择器

举例 意义
img[alt] 选择有alt属性的img标签
img[alt=’故宫’] 选择有alt属性是故宫的img标签
img[alt^=’北京’] 选择有alt属性以北京开头的img标签
img[alt$=’夜景’] 选择有alt属性以夜景结尾的img标签
img[alt*=’美’] 选择有alt属性中含有美字的img标签
img[alt~=’手机拍摄’] 选择有alt属性中有空格隔开的手机拍摄独立部分img标签
img[alt|=’参赛作品’] 选择有alt属性以’参赛作品-‘开头的img标签

伪元素

::before 创建一个伪元素，其将成为匹配选中的元素的第一个子元素，必须设置content属性表示其中的内容a::before{content:'⭐'}

::after 创建一个伪元素，其将成为匹配选中的元素的最后一个子元素，必须设置content属性表示其中的的内容a::after{content:'♥'}

::selection css伪元素应用于文档中被用户高亮的部分（使用鼠标圈选的部分）box::selection{color:red}

::first-letter 选中某元素中(必须是块级元素)第一行的第一个文字 box::first-letter{ font-size:2rem }

::first-line 选中某元素中(必须是块级元素)第一行的全部文字 box::first-line{ text-decoration:underline }

层叠性和权重

层叠性：多个选择器可以同时作用于同一个标签，效果叠加。

如果多个选择器的属性有冲突，css有处理冲突的规则：id权重 > class权重 > 标签权重

复杂选择器可以通过(id的个数，class的个数，标签的个数)的形式，计算权重

使用 !important 提升权重

text-indent

text-indent 定义首行文本内容缩进。例：text-indent:2rm;

继承性

文本相关的属性普遍具有继承性，只需要给祖先标签设置，即可在后代所有标签中生效。

在继承的情况下，选择器权重计算失效，而是就近原则

overflow

overflow:hidden 表示溢出隐藏，溢出盒子边框的内容将会被隐藏

overflow:scroll 表示溢出使用滚动条显示，溢出盒子边框的内容使用滚动条显示。

阴影

border-shadow: inset 10px 20px 30px 40px rgba(0,0,0,.4)
内阴影 x y 模糊量 阴影延申 颜色和透明度

可以设置多层阴影叠加，用逗号隔开就可以

背景

background-image: url() 设置背景图片

background-repeat 属性用来设置背景的重复的模式。值：repeat = xy均平铺，repeat-x = x平铺， repeat-y = y平铺， on-repeat = 不平铺

background-size 属性用来设置背景图片的尺寸，可以使用百分比，auto设置等比例值。background-size:auto。 contain和cover 是连个特殊的值，contain表示将背景图片智能改变尺寸以容纳到盒子中，cover表示将背景图片只能改变尺寸以撑满盒子

background-clip 用来设置元素的背景裁切到哪个盒子，兼容到IE9

值 意义
border-box 背景延申至边框（默认值）
padding-box 背景延申至内边（padding），不会绘制到边框处（仅在dotted、dashed边框可察觉）
content-box 背景被裁剪至内容区

background-attachment 属性决定背景图像的位置是在视口内固定，或者随着包含它的区块滚动

值 意义
fixed 自己滚动条不动，外部滚动条不动
local 自动滚动条动，外部滚动条动
scroll 自动滚动条不动，外部滚动条动（默认值）

background-position 属性可以设置背景图片出现在盒子的什么位置。background-position:100px 200px;。可以使用top、bottom、center、left、right描述图片出现的位置，可以使用bottom center让图片显示在中间底部。将多个小图标合并制作到一张图片上，使用background-position属性单独显示其中一个，叫做css雪碧图

background 综合属性，使用：background: white url() on-repeat center center，按顺序是，背景颜色、背景图片、背景重复、背景位置

盒子的background-image属性可以用radial-gradient() 形式创建径向渐变背景。background-image:radial-gradient(50% 50%, red, blue, yellow)可以设置多个颜色

旋转变形

transform 属性的值设置为rotate()，即可实现旋转变形。transform: rotate(45deg); 旋转角度，若角度为正，则顺时针旋转，否则逆时针方向旋转

将transform属性的值设置为scale(3)，即可实现缩放变形。transform:scale(3); 当数值小于1时，表示缩小元素，大于1表示放大元素

将transform属性的值设置为skew()，即可实现斜切变形。transform:skew(10deg,20deg); x斜切角度，y斜切角度

将transform属性的值设置为translate()，即可实现位移变形，和相对定位非常像，位移也会在原来的地方留坑，形影分离。transform:translate(200px,300px)

transform-origin 属性的值可以设置自定义变换原点。transform-origin:0 0;

3D旋转

transform属性的值设置为rotateX()或者rotateY()，即可实现绕横轴、纵轴旋转。transform: rotateX(30deg);

perspective 属性用来定义透视强度，可以理解为人眼到舞台的距离，单位是px

当元素进行3D旋转后，即可继续添加translateX()、translateY()、translateZ()属性让元素在空间进行移动

js

设置自适应字体

字体设置为页面的十分之一，如果页面过大的话就会导致字体也会变的很大，可以设置一个最大值，超过这个最大值字体就使用一个固定的值
